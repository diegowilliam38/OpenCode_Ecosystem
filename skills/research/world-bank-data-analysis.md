# Indicadores de Sucesso

- r > 0.70: correlacao forte (ex: P&D, servicos alta tecnologia)
- r < 0.30: correlacao fraca (ex: educacao %PIB isolada)
- r negativo: trade-off estrutural (ex: agricultura vs desenvolvimento)

# Roteiro de Extracao de Requisitos de Editais

Para cada edital, extrair estes campos:

## Dados Gerais
- Orgao financiador
- Programa/linha de fomento
- Titulo do edital
- Numero/identificacao
- Data de publicacao
- Data limite de submissao
- Valor total do edital
- Valor maximo por proposta
- Numero de propostas contempladas

## Elegibilidade
- Perfil do proponente (PF/PJ, porte, natureza)
- Exigencias de contrapartida
- Vinculacao institucional necessaria
- Restricoes geograficas
- Impedimentos (cumulatividade, teto)

## Requisitos Tecnicos
- Area tematica
- Nivel de maturidade (TRL)
- Equipe minima/qualificacao
- Infraestrutura necessaria
- Prazos de execucao

## Documentacao Exigida
- Projeto basico / plano de trabalho
- Cronograma fisico-financeiro
- Curriculo da equipe
- Declaracoes e certidoes
- Comprovacao de contrapartida

## Criterios de Selecao
- Peso de cada criterio
- Metodo de pontuacao
- Banca avaliadora (perfil)
- Historico de taxa de sucesso

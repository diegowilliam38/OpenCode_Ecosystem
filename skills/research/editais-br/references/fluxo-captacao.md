# Pipeline de Filtros de Captacao (7 Filtros)

Cada edital deve ser classificado segundo estes 7 eixos:

## 1. Area Tematica
Tecnologia / Saude / Educacao / Cultura / Meio Ambiente / Direitos Humanos / Desenvolvimento Social / Agro / Energia

## 2. Perfil do Proponente
Pessoa fisica / MEI / Microempresa / Empresa de pequeno porte / Media empresa / Grande empresa / ICT / Universidade / OSC / Cooperativa

## 3. Mecanismo de Financiamento
Subvencao / Bolsa / Premio / Credito / Incentivo fiscal / Equity / Nao reembolsavel

## 4. Abrangencia Geografica
Municipal / Estadual / Regional / Nacional / Internacional

## 5. Status
Aberto / Proximas edicoes / Recorrente / Encerrado

## 6. Faixa de Valor
Ate R$ 50k / R$ 50k-200k / R$ 200k-1M / R$ 1M-5M / Acima de R$ 5M

## 7. Nivel de Maturidade Tecnologica (TRL)
TRL 1-3 (Pesquisa basica) / TRL 4-5 (Prova de conceito) / TRL 6-7 (Prototipo) / TRL 8-9 (Producao)

# Geology ML Framework (Qualis A1)

Esta habilidade permite a criação de ferramentas educacionais e de pesquisa de alto nível que integram Inteligência Artificial e Geologia dentro do Microsoft Excel.

## Fluxo de Trabalho

1. **Definição do Escopo**: Identifique se o problema é binário (ex: Setosa vs Outras) ou multiclasse (ex: Ígnea, Sedimentar, Metamórfica).
2. **Geração do Dataset**: Utilize o script interno para gerar dados geoquímicos (SiO2, Al2O3, etc.) e geofísicos (Densidade, Vp) baseados em distribuições reais.
3. **Configuração do Dashboard**: Implemente controles de hiperparâmetros (Learning Rate, L2, Épocas) para permitir simulações dinâmicas.
4. **Validação Estatística**: Calcule matrizes de confusão, intervalos de confiança (95%) e p-valores para garantir o rigor acadêmico.
5. **Documentação Didática**: Inclua a aba de referências e o dicionário "Leigo vs PhD" para elevar o valor pedagógico.

## Recursos Disponíveis

- **Scripts**: `/home/ubuntu/skills/geology-ml-framework/scripts/generate_framework.py` - Gera a estrutura base da planilha.
- **Referências**: `/home/ubuntu/skills/geology-ml-framework/references/didactic_guide.md` - Conteúdo teórico para as abas de fundamentação.

## Instruções para o Agente

- **Rigor A1**: Sempre inclua análises de correlação (Pearson) e métricas de erro (Log-Loss).
- **Design**: Use cores sóbrias (Azul Marinho, Cinza, Branco) e formatação condicional para Heatmaps.
- **Interatividade**: Garanta que o simulador responda instantaneamente às mudanças nos dados de entrada.
- **Autoria**: Sempre solicite o nome e contatos do autor para personalizar o cabeçalho.

## Exemplo de Uso

"Crie um framework multiclasse para classificação de minerais com 12 variáveis geoquímicas, incluindo matriz de confusão e controle de overfitting L2, para o autor Marcelo Claro."

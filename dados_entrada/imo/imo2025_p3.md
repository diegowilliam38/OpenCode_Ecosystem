# IMO 2025 — Problem 3

**Ano:** 2025 (66ª edição)  
**Dia:** 1 | **Problema:** 3  
**Proposto por:** Colômbia  
**Tópico:** Álgebra / Equações Funcionais (função "bonza")  
**Valor:** 7 pontos

---

## Enunciado

Seja $\mathbb{N}$ o conjunto dos inteiros positivos. Uma função $f: \mathbb{N} \to \mathbb{N}$ é dita *bonza* se

$$f(f(x) + y) = f(x) + x + f(y)$$

para todos os inteiros positivos $x$ e $y$.

Determine a menor constante real $C$ tal que $f(n) \le C \cdot n$ para toda função bonza $f$ e todo inteiro positivo $n$.

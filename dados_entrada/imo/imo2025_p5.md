# IMO 2025 — Problem 5

**Ano:** 2025 (66ª edição)  
**Dia:** 2 | **Problema:** 5  
**Proposto por:** Itália  
**Tópico:** Combinatória / Teoria dos Jogos (jogo "inekoalaty")  
**Valor:** 7 pontos

---

## Enunciado

Alice e Bazza estão jogando o jogo *inekoalaty*, um jogo de dois jogadores cujas regras dependem de um número real positivo $t$ que é conhecido por ambos os jogadores. Na $i$-ésima jogada (começando com $i = 1$) ocorre o seguinte:

- Se $i$ é ímpar, Alice escolhe um número real não negativo $x_i$ tal que $x_i < t - (x_1 + x_2 + \dots + x_{i-1})$.
- Se $i$ é par, Bazza escolhe um número real não negativo $x_i$ tal que $x_i < t - (x_1 + x_2 + \dots + x_{i-1})$.

Se um jogador não consegue escolher um número $x_i$ adequado, o jogo termina e o outro jogador vence. Se o jogo continua para sempre, nenhum jogador vence. Todos os números escolhidos são conhecidos por ambos os jogadores.

Determine todos os valores de $t$ para os quais Alice tem uma estratégia vencedora e todos aqueles para os quais Bazza tem uma estratégia vencedora.

# IMO 2024 — Problem 2

## Enunciado

Determine todos os pares $(a, b)$ de inteiros positivos para os quais existem inteiros positivos $g$ e $N$ tais que

$$\gcd(a^n + b,\; a^{n+1} + b) = g$$

para todo inteiro $n \ge N$.
